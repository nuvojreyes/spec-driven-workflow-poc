# Steps for AI setup

1. Create .github folder in the root of the repository if it does not exist.
2. Inside the .github folder, create a file named copilot-instructions.md.
3. Create agents and prompts folders inside .github/
4. Create files for the specific agents and prompts you want to use inside the agents/ and prompts/ folders.
5. Open Copilot Chat and then go to settings. Click on 'Generate Chat Instructions' button, it will generate a template for copilot-instructions.md file.
